# SETUP GUIDE

### PHP version

```sh
"php": "^8.2",
"laravel/framework": "^11.31"
```

### Required php-extentions

```sh
apt install php8.2
```

php.ini
```
extension=pdo_mysql
```

### MySQL

add UTC timezone in my.ini
```sh
[mysqld]
default-time-zone = '+00:00'
```

### node-npm

```sh
$ npm -v
10.9.0

$ node -v
v22.12.0

$ nvm -v
0.39.3
```
